<h1>公主连接自动批量买药水的脚本</h1>  

使用方法  
打开公主连接dmm版，进入商店界面  
在config.txt中输入想要购买的次数（写一个大于0的整数不要乱写别的）  
在config.txt中根据自己网络状况输入延时系数（有网络通信部分的等待时间会乘上这个系数）  
选择是否买药或石头  
以管理员身份运行potionbuyer.exe  
不要进行其他操作等他自动买完。。。  
希望中途停下来时摁住s直到脚本停止

注意：需要以管理员身份运行  
          只在自家电脑上dmm端可以拉到的最大窗口进行了测试，没有测试在不同环境下的兼容性  
          根据网络情况可能出现失败（如果通信时间大于预设休眠时间的话。。。）
